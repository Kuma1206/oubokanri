# ②課題内容（どんな内容か）
・新規登録情報をデータベースに保存  
・ログイン情報をデータベースでidとuser_idを紐付けしアカウント内の情報とログイン情報を連携
・アカウント毎にデータベースに情報を保存  
・My個人情報の管理　のみ保存ができます。他はこれからです。
